## 武器

玩家进行游戏时会携带一把初始武器，并且在5级，15级，25级时可以获得新的武器。武器的解锁需要完成对应的里程碑内容

武器词条会分成四个类别

1. 伤害类型：动能，燃烧，电击，急冻，电浆，腐蚀
2. Family：轻型，中型，重型，投掷物，建造物
3. Weapon type：发射物，爆炸，无人机，哨戒炮
4. Firing type：精密，散射，范围，射线，长时
