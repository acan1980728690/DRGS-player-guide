欢迎访问DRGS玩家攻略站！此网站由个人独立维护，爱心群友参与编写内容，与官方并没有关系，也不会加入商业性内容，内容更新与校对依赖于站长及热心玩家的志愿贡献。如有疏漏或错误，敬请谅解，也欢迎您在群内或在github内提出问题，帮助我们共同完善。（晚点加入评论功能）

本站大部分内容（包括但不限于基础数据、机制说明等）参考或翻译自游戏的[DRGS官方wiki](https://deeprockgalactic.wiki.gg/wiki/Survivor:Main)。**强烈建议**具备英文阅读能力的玩家直接访问官方 Wiki，以获取最权威、最新、最完整的资讯。

除参考官方资料外，本站大多数页面均包含由社区玩家原创撰写的攻略内容。这些内容仅供参考

本站部分内容为非官方翻译，可能存在翻译误差、术语不一致或与官方中文版本存在差异的情况。如有差别，请以游戏内实际内容为准。
